# Soal 2
Karyawan honorer di perusahan XXX digaji berdasarkan jumlah jam nkerjanya selama satu minggu. Upah per jamnya adalah Rp. 2.000,-. Bila jumlah jam kerja selama satu minggunya lebih besar dari 48 jam, maka sisanya dianggap jam lembur dengan upah per jam lemburnya adalah Rp. 3.000,-.

Buatlah form untuk memasukkan jumlah jam kerja selama satu minggu seorang karyawan. Setelah form disubmit, maka akan tampil jumlah upah yang diterima karyawan tersebut. Gunakan script PHP untuk membuat hal ini.
